# V2Ray 使用指南

## 1. 快速使用（复制链接与一键导入）

您可以直接通过控制台或者 SSH 登录服务器后，执行以下命令获取您的连接凭据：

```bash
cat /opt/cloud/v2ray/credentials.txt
```

根据输出的内容，快速完成导入：

**第 1 步：复制分享链接**

在控制台输出中，选中并复制一整段长链接字符串。请确从 `vless://`（或您生成的其他协议）开头，一直复制到最后（例如直到末尾的 `...#V2Ray-Reality` 等标识符）。

**第 2 步：一键导入**

根据您的操作系统，按照下方对应的指示将复制好的链接导入至客户端：

- **Windows (v2rayN)**
  打开应用，在键盘上直接按下 `Ctrl + V`，或者点击上方“服务器”菜单并选择“从剪贴板导入批量 URL”。

- **macOS (V2RayX / V2Free / Clash)**
  点击应用在顶部菜单栏的对应图标，选择“从剪贴板导入配置”或“添加剪贴板”。

_(移动端等其他平台的详细步骤，请参见下方的[客户端配置](#4-客户端配置)章节。)_

如果您的特定场景必须获取原始的底层参数：

```bash
cat /opt/cloud/v2ray/config/config.json
```

## 2. 关键目录说明

| 目录/文件                             | 描述                                         |
| ------------------------------------- | -------------------------------------------- |
| `/opt/cloud/v2ray/config/config.json` | V2Ray 的核心配置文件（包含入站、出站规则）。 |
| `/opt/cloud/v2ray/credentials.txt`    | 包含节点登录、链接分享等明文信息的凭据文件。 |
| `/var/log/v2ray/`                     | V2Ray 的访问与错误日志存放目录。             |

## 3. 常用操作

进入 `/opt/cloud/v2ray` 目录，您可以使用以下内置的快捷管理脚本（视具体部署方式而定，您也可以使用 `docker` 命令进行管理）：

| 操作     | 命令                                                  |
| -------- | ----------------------------------------------------- |
| 启动服务 | `/opt/cloud/v2ray/start.sh`                           |
| 停止服务 | `/opt/cloud/v2ray/stop.sh`                            |
| 重启服务 | `/opt/cloud/v2ray/restart.sh`                         |
| 查看状态 | `/opt/cloud/v2ray/status.sh`                          |
| 查看日志 | `/opt/cloud/v2ray/logs.sh` 或 `docker logs v2ray-app` |

## 4. 客户端配置

我们为您推荐以下覆盖各个主流操作系统的优秀客户端程序，以获取最佳体验。

### 💻 Windows 平台：v2rayN

**v2rayN** 是目前 Windows 上最稳定且功能覆盖最广的图形化客户端。

- **下载地址**：[GitHub Releases - 2dust/v2rayN](https://github.com/2dust/v2rayN/releases)
  - _注：新手建议下载 `v2rayN-With-Core.zip`，内含运行内核，解压即用。_
- **安装步骤**：
  1.  解压下载的压缩包到非系统盘文件夹。
  2.  双击运行 `v2rayN.exe`（如提示需要 .NET Desktop Runtime，请根据弹窗指引安装）。
- **快捷配置指南**：
  1.  **复制链接**：复制上述控制台输出的单节点分享链接（通常以 `vless://` 为开头）。
  2.  **一键导入**：打开 v2rayN 主界面，在键盘上直接按下 `Ctrl + V`，或者点击上方“服务器”菜单并选择“从剪贴板导入批量 URL”。
  3.  **选中节点**：在节点列表中按 `Enter` 键或右键将其设为活动节点。
  4.  **启动代理**：右键点击右下角系统托盘的软件图标，选择 **系统代理** -> **自动配置系统代理**。此时图标变为紫色即代表成功。

### 📱 Android 平台：v2rayNG

**v2rayNG** 延续了 v2rayN 的易用性，是安卓端的首选。

- **下载地址**：[GitHub Releases - 2dust/v2rayNG](https://github.com/2dust/v2rayNG/releases) 或 Google Play 商店。
- **快捷配置指南**：
  1.  **一键导入**：复制链接后，打开 v2rayNG 主界面，点击右上角的 **“+”** 图标。
  2.  **从剪贴板添加**：在弹出的菜单中选择 **“从剪贴板导入”**，应用会自动识别并添加此节点。
  3.  **选择与连接**：点击列表中刚导入的节点条目，然后点击右下角的 **圆形“V”图标** 启动代理。如果提示连接权限，请点击“允许”。

### 🍎 macOS 平台：V2Box / V2rayU

由于使用系统生成的单节点分享长链接，我们更建议 macOS 用户使用原生支持剪贴板一键导入的便捷客户端：

- **下载推荐**：
  - **V2Box**：推荐在 Mac App Store 中搜索并免费下载（原生支持苹果 M 芯片及 Intel）。
  - **V2rayU**：经典开源客户端，可前往 [GitHub Releases](https://github.com/yanue/V2rayU/releases) 下载。
- **快捷配置指南（以 V2Box 为例）**：
  1.  **一键导入**：复制链接后，打开 V2Box。进入底部的 **“配置” (Configs)** 页面，点击右上角 **“+”** 然后选择 **“从剪贴板导入” (Import from clipboard)**。
  2.  **启动代理**：切换到 **“首页” (Home)**，轻点居中的大按钮或滑动底部开关即可连接。
- **快捷配置指南（以 V2rayU 为例）**：
  1.  **一键导入**：打开应用后，点击屏幕顶部系统菜单栏的 `V2rayU` 图标，直接选择 **“Import from Clipboard” (从剪贴板导入)**。
  2.  **启动代理**：在菜单栏图标中选择 **Turn V2ray-Core On**，并在模式中选择 **Global Mode** 或 **Pac Mode** 即可。

### 🍏 iOS 平台：V2Box

iOS 环境下推荐使用免费且支持多协议的 **V2Box**，它的操作逻辑极其简单，非常适合新人一键导入。

- **下载地址**：在 App Store 中搜索 `V2Box` 并获取。
  - _注：如果在中国区 App Store 无法搜索到应用，请使用海外区域的 Apple ID 登录下载。_
- **快捷配置指南**：
  1.  **一键导入**：复制链接后，打开 V2Box。点击底部菜单栏的 **“配置” (Configs)**，然后点击右上角的 **“+”** 号，选择 **“从剪贴板导入” (Import from clipboard)**。
  2.  **启动代理**：切换回底部的 **“首页” (Home)** 标签，轻点屏幕中央的连接大按钮，或向上滑动底部的开关。
  3.  **授权**：首次连接时，系统会提示需要添加 VPN 配置，请点击“允许” (Allow) 并使用密码或 FaceID 授权即可完成配置。

### ⚠️ 通用注意事项

1.  **模式选择**：建议所有平台优先选择 **“绕过大陆 (Bypass Mainland)”** 或 **“规则模式”**。这样国内流量不走代理，速度更快且省流量。
